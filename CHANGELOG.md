# Changelog

All notable changes to this project are documented in this file.

## 1.0.0

- Add an **Open in Adobe Photoshop** action to the DayZ PAA Preview toolbar.
- Add matching Explorer context-menu and Command Palette actions for local PAA files.
- Discover compatible local Adobe Photoshop installations and support an explicit machine-scoped executable path.
- Integrate through the public DayZ PAA Preview companion-action API.
- Provide English and German manifest localization.
